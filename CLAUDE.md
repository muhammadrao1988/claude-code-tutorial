# Claude Code Tutorial Project

This is a tutorial repo that teaches Claude Code concepts through simple, real-life examples.

## Project Structure

- `01-memory/` through `08-orchestration/` — individual tutorials
- `backlog/` — upcoming tutorial topics (pending/) and completed ones (completed/)
- `specs/` — spec documents for each completed tutorial
- `.claude/` — agents, commands, skills that manage this repo

## Development Workflow

1. Check backlog: `/backlog`
2. Scaffold new tutorial: `/new-tutorial`
3. Write content using tutorial-writer agent
4. Publish: `/publish`

## Writing Rules

- Every tutorial has two examples: non-tech + developer
- Use plain English, short paragraphs
- All code blocks must be copy-pasteable
- Structure: What -> Why -> Setup -> Try It
- No emojis in tutorial content
- Keep each tutorial README under 200 lines

## Git

- Commit often with descriptive messages
- Push to muhammadrao1988/claude-code-tutorial
- 2 tutorials per day maximum
