Agent Teams let you coordinate multiple Claude Code sessions working together. One session leads, others work as teammates with shared tasks and direct messaging. Requires enabling via settings.json. Tutorial should clearly explain the difference between subagents (Tutorial 03) and agent teams -- subagents run inside one session, agent teams are separate sessions that talk to each other.
